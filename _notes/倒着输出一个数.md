[[C语言学习]]

只是倒着输出，即可以
	1）生成一个倒着的数，然后输出
	2）按照倒的顺序一个一个输出

思路1的一种：

```C
int upturned_n;
while (n){
	upturned_n = upturned_n*10 + n%10;
	n /= 10; 
}
```

用一道题展示思路2
《将一个十进制的非负整数转化成二进制输出》
思路2的两种：
1)递归
```C
//先展示一个错误版本
void dectobin( int n ){
    if (n){
        printf("%d",n%2);
        doctobin(n/2);
    }
}                       //这个生成的是倒的，比如10 生成的就是0101
//当时处理起来不太好闹
//其实换一下思路就好了
void dectobin( int n ){
    if (n){
	    doctobin(n/2);
        printf("%d",n%2);
    }
} 
//处理一下n=0的情况
void dectobin( int n ){
    if (n==0){          //n＝0的情况
        printf("0");
        return;
    }
    if (n){
        if (n/2)         //防止之后再执行上面n=0情况，影响
            dectobin(n/2);
        printf("%d",n%2);
    }
}
```
2)用数组循环
```C
void decimalToBinary(int num) {
    int binaryNum[32];
    int i = 0;

    while (num > 0) {
        binaryNum[i++] = num % 2;
        num /= 2;
    }

    for (int j = i - 1; j >= 0; j--)
        printf("%d", binaryNum[j]);
}
```
