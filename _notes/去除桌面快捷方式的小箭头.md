要在Windows 11中去除桌面快捷方式图标上的小箭头，可以通过修改注册表来实现。请按照以下步骤操作：

### 注意：修改注册表有一定风险，请在操作前备份好注册表，避免出现问题。

### 方法一：手动修改注册表
1. **打开注册表编辑器**
   - 按下 `Win + R` 键，输入 `regedit`，然后按下回车键。

2. **定位到以下路径**
   在注册表编辑器中，定位到以下路径：
   ```
   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer
   ```

3. **创建新的键**
   - 右键点击 `Explorer` 文件夹，选择 **新建** -> **项**，将其命名为 `Shell Icons`。

4. **创建新的字符串值**
   - 在 `Shell Icons` 文件夹中，右键点击右侧空白处，选择 **新建** -> **字符串值**，将其命名为 `29`。

5. **修改字符串值的数据**
   - 双击新建的 `29` 字符串值，在“数值数据”栏中输入以下内容：
     ```
     %windir%\System32\shell32.dll,-50
     ```
   - 然后点击“确定”。

6. **重启电脑**
   修改完成后，关闭注册表编辑器并重启电脑，桌面快捷方式的小箭头就会消失了。

### 方法二：使用第三方软件
如果不想手动修改注册表，也可以使用一些第三方工具，例如 **Winaero Tweaker**。这是一个免费的工具，可以帮助你轻松去掉快捷方式图标的小箭头。 ^68497d

#### 使用 Winaero Tweaker 的步骤：
1. 下载并安装 [Winaero Tweaker](https://winaero.com/winaero-tweaker/).
2. 打开 Winaero Tweaker，找到左侧菜单中的 **Shortcut Arrow**。
3. 在右侧选择 **No arrow**，然后点击 **Apply Changes**。
4. 重启电脑以使更改生效。

这样，你就可以通过简单的几步操作去除桌面快捷方式图标上的小箭头。
