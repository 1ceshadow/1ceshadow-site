在做CS50的week2的substitution作业时，看到一人分享的解法
其中，判断传入参数 key 是否有重复的字符

```C
int exist[26] = {0};
string key = argv[1];

for (int i = 0; i < 26; i++) {
    exist[toupper(key[i]) - 'A']++;
}
    
for (int i = 0; i < 26; i++) {
    if (exist[i] != 1) {  // 这里应该考虑== 0 的情况
        printf("Key must not contain repeated characters.\n");
        return 1;
    }
}
```
