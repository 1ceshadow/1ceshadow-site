[[C语言学习]] #CS50 

数字
--
+ 计算机常用8位表示数字，`00000101` = 5。

---
数字用0和1的二进制表示，字母也可以用
--
+ ***用ASCII***
+ 由于表示数字和字母的 1 和 0 之间存在重叠，因此创建了 _ASCII_ 标准以将特定字母映射到特定数字。
![[Pasted image 20241029230850.png]]
+ 如果您收到一条短信，则该消息下的二进制文件可能表示数字 72、73 和 33。将这些映射到 ASCII，您的信息将如下所示：
```bash
H   I   !
72  73  33
```
---

Unicode
--
+ 但ASCII表示的字符太少了，像咱们国家常用字就有3000字，根本不够用
+ 于是产生了Unicode码，
+ Unicode有110 万个码点，如此之大，以至于目前已定义的约有 17 万个，即 15%
+ 科学家还规定了特殊字符和表情符号，甚至为表情符号增添不同的肤色。

---

Representation
--
+ 0和1也可以表示颜色。
+ 红色(red)、绿色(green)和蓝色(blue)（称为 `RGB）`是三个数字的组合。
	![[Pasted image 20241029234227.png]]
+ 以之前使用的 72、73 和 33 为例，
	- 通过文本显示 `HI！`，
	- 图像阅读器会将它解释为浅黄色阴影。红色值为 72，绿色值为 73，蓝色值为 33。 ![[Pasted image 20241029234402.png]]
----
此外，0 和 1 可用于表示图像、视频和音乐！

+ 图像只是 RGB 值的集合。
+ 视频是存储在一起的许多图像的序列，就像翻书一样。
+ 音乐可以通过 MIDI 数据来表示。
