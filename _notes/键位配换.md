### Debian
[Arch Wiki](https://wiki.archlinux.org/title/Xorg/Keyboard_configuration#One-click_key_functions)
查看当前系统图形显示协议类型，输入`echo $XDG_SESSION_TYPE`，如果显示 `x11`则说明可用。
```bash
~$ echo $XDG_SESSION_TYPE
x11
```

1. 交换 ctrl 和 caps  
修改 `/etc/default/keyboard`，将 `XKBOPTIONS` 修改为以下内容
```
XKBOPTIONS="ctrl:swapcaps"
```
2. 安装 xcape
```bash
sudo apt install xcape
```
2. 自启动  
新增一个自启动文件 `sudo vim ~/.config/autostart/xcape.desktop`
```bash
[Desktop Entry]
Name=Xcape
Exec=/usr/bin/xcape -e 'Control_L=Escape'
Terminal=false
Type=Application
StartupNotify=true
```

### KDE(Wayland)
#### xcape
倒也可以用
#### 用[evremap](https://github.com/wez/evremap/tree/master) 
- AUR 安装
- 创建配置文件 `/etc/evremap.toml` 
- 重新加载 systemd 配置 `sudo systemctl daemon-reload`
```
sudo systemctl enable evremap.service
sudo systemctl start evremap.service
```

`C:\Users\<UserName>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
