[[C语言学习]] #CS50 

+ 首先，*预处理*是将代码中由 `#` 表示的头文件（如 `#include <cs50.h>`）有效地复制并粘贴到文件中。在此步骤中，`cs50.h` 中的代码将复制到您的程序中。同样，正如您的代码包含 `<stdio.h> #include`，计算机上某个位置的 `stdio.h` 中包含的代码也会被复制到您的程序中。此步骤可以可视化如下：
```c
string get_string(string prompt);
int printf(string format, ...);

int main(void)
{
    string name = get_string("What's your name? ");
    printf("hello, %s\n", name);
}
```

+ 第三，*汇编*涉及编译器将汇编代码转换为机器代码。此步骤可以可视化如下：![[Pasted image 20241029235218.png]]
+ 最后，在*链接*步骤中，您包含的库中的代码也会转换为机器代码并与您的代码组合。然后输出最终的可执行文件。![[Pasted image 20241029235236.png]]
